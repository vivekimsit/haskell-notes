haskell-notes
=============

Haskell notes for https://courses.edx.org/courses/DelftX/FP101x/3T2014/info

| Mathematics  | Haskell |
| ------------- | ------------- |
| f(x)  | f x  |
| f(x, y)  | f x y  (or x \`f\` y)|
| f(g(x))  | f (g x)  |
| f(x, g(y))  | f x (g y)  |
| f(x)g(y)  | f x * g y  |
